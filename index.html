I collaborated with the Demicube team to build a website for their Future Fest Virtual Reality Application. Future Fest simulates Music Festival environments, allowing their artists to DJ shows, their users to attend shows and interact with other concert attendees. The website, aims to support this enviroment by enabling users to create accounts, upload assets (music, video, and image files), edit those assets, and share their work with others. Using React, HTML5/CSS/Javascript, I created a user interface that interacts with the Future Fest API to store uploads, retrieve stored data, enable downloads, allow edits, play/pause audio and video files, and give users display layout options.

ShadowPlay is an Application that allows the user to livestream their desktop and engage in conversation through a chat feature. This Nodejs, React HTML/CSS/Javascript application implements browser extensions that gives the user's broswer permission to acces their desktop camera stream. The Nodejs server uses Web Sockets to communicate with the React fron-end, enabling camera stream and chat streams to be exchanged easily.

ProfileCard is an application that enables users to create an account, sign in to their accounts, store/edit their profile information, and view other user information. The MERN (MongoDB, Express, React, Nodejs) technology stack was used to create an API server that connects to the MongoDB database, make calls to that API through React, and display retrieved API json objects.
